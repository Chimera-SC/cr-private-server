## Setup
1. ```mvn clean package``` or ```mvn clean install```     
2. ```java -jar target\private-server-1.0-SNAPSHOT.jar``` or ```java --add-opens java.base/java.nio=ALL-UNNAMED -jar target\private-server-1.0-SNAPSHOT.jar```
